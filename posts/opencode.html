¿Qué es OpenCode?

OpenCode es un agente de programación de código abierto con licencia MIT, diseñado para operar principalmente en la terminal (CLI/TUI), aunque también cuenta con aplicaciones de escritorio y extensiones para diversos IDEs. Con más de 149.000 estrellas en GitHub, se ha consolidado como una de las herramientas más populares y completas para el desarrollo asistido por inteligencia artificial.

¿Cómo funciona OpenCode?

A diferencia de un simple chatbot, OpenCode opera mediante un "bucle agéntico", lo que le permite analizar archivos, ejecutar comandos en la shell, escribir código y realizar pruebas de forma autónoma hasta completar una tarea. Su flujo de trabajo se divide en dos estados fundamentales que el usuario puede alternar con la tecla Tab:
  • Modo Plan: Es un modo de solo lectura ideal para explorar bases de código desconocidas, realizar análisis de arquitectura o diseñar estrategias sin riesgo de modificar archivos.
  • Modo Construir: Es el modo de acción predeterminado, donde el agente tiene permisos de escritura para editar archivos y ejecutar comandos reales en el sistema.

Características Técnicas Clave

OpenCode destaca por su capacidad de entender profundamente el contexto de un proyecto mediante el comando /init, el cual genera un archivo AGENTS.md que resume la estructura y convenciones del código base. Además, integra tecnologías avanzadas como:
  • LSP (Language Server Protocol): Para recibir diagnósticos y errores de compilación en tiempo real directamente del lenguaje de programación utilizado.
  • MCP (Model Context Protocol): Que permite conectar la IA con herramientas externas como bases de datos, navegadores web o servicios en la nube.
  • Skills y Plugins: Conocimientos empaquetados y extensiones que modifican el comportamiento del agente según las necesidades del desarrollador.

Diferencias con otros Modelos

Mientras que herramientas como Claude Code están ligadas estrictamente al ecosistema de Anthropic, OpenCode es agnóstico del proveedor. Esto significa que soporta más de 75 proveedores de modelos, permitiendo al usuario elegir entre modelos premium como GPT-4 o Claude 3.5 Sonnet, o incluso modelos locales que se ejecutan totalmente fuera de línea mediante Ollama o LM Studio. Finalmente, OpenCode ofrece una mayor soberanía de datos y control de costos. Bajo una filosofía "local-first", el código permanece en la máquina del desarrollador y permite aprovechar suscripciones existentes de servicios como GitHub Copilot o ChatGPT Plus, evitando pagos adicionales por tokens si se utilizan configuraciones locales.
